# Restaurant Menu Allergen Checker

## Overview

Restaurant Menu Allergen Checker is a simple AI-inspired application that helps users identify dishes that may contain allergens.

The application retrieves ingredient information from a local knowledge base and compares it against the user's allergies to generate a structured safety report.

## Features

- Upload menu (.txt)
- Paste restaurant menu
- Enter allergies
- Detect risky dishes
- Show safe dishes
- Download report
- Retrieval-based knowledge lookup

## Technologies

- Python
- Streamlit
- Local Knowledge Base
- RAG-inspired Retrieval

## Run

pip install -r requirements.txt

streamlit run app.py
